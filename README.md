# Mini projet Cypress - SauceDemo

Ce projet contient un exemple minimal de tests Cypress pour le site SauceDemo.

## Installation

```bash
npm install
```

## Scripts npm

- Ouvrir Cypress en mode interactif :
  ```bash
  npm run test:open
  ```

- Exécuter les tests en ligne de commande :
  ```bash
  npm run cypress:run
  ```

## Tests inclus

- Login valide
- Login invalide
- Ajout d’un article au panier avec vérification du badge
